## Installation

To install the Eternal Calendar, drop the `build/libs/eternal-calendar-1.0.0.jar` file into the `minecraft/mods/` folder.

### Requirements
- **Fabric Loader**: 0.16.5+
- **Fabric API**: 0.116.11+1.21.1 for MC 1.21.1
